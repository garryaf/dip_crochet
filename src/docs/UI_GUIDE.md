# UI/UX Guide - Cotcret

## Design System
- **Font**: Outfit (Sans-serif) for headlines, Geist Sans for body.
- **Colors**:
  - `Primary`: #ff8fb1 (Soft Pink) - Used for primary actions and highlights.
  - `Secondary`: #6ebfb5 (Mint Green) - Used for secondary actions and "Graduation" themes.
  - `Background`: #fffbf9 (Cream) - Warm, friendly base color.
  - `Shadow`: `0 10px 30px -10px rgba(0, 0, 0, 0.05)` - Very soft, premium depth.

## Key UI Components
### 1. Rounded Cards
All cards use `rounded-[2.5rem]` or `rounded-[3rem]` to maintain a friendly, soft feel consistent with crochet materials.

### 2. Glassmorphism
Used for navigation bars and overlay panels (`.glass-card`). It provides a premium, modern feel without cluttering the background.

### 3. Animations
- **Framer Motion**: Used for entrance animations on the landing page.
- **Three.js Float**: The 3D models gently "breathe" (float) to feel alive.
- **Hover Effects**: Subtle scaling (+5%) on interactive components.

## Branding Voice
The copy should always be:
- Emotional
- Gentle
- First-person when describing characters ("I love...", "Meet me at...")
- Focused on "memories" and "storytelling".
